# Swag Labs – Test Scenarios

## TS-01: Successful Login
Verify that a registered user can log in with valid credentials and is redirected to the Products page.

## TS-02: Invalid Login Attempt
Verify that appropriate error messages are displayed when invalid username or password is entered.

## TS-03: Locked User Login
Verify that a locked user cannot log in and receives the correct error message.

## TS-04: View Products
Verify that all products are displayed correctly with name, image, description, and price on the Products page.

## TS-05: Add Product to Cart
Verify that a user can add a product to the cart and the cart badge count increases accordingly.

## TS-06: Remove Product from Cart
Verify that a user can remove a product from the cart and the cart badge count decreases.

## TS-07: Product Sorting
Verify that products can be sorted by name (A–Z, Z–A) and price (low to high, high to low).

## TS-08: Burger Menu Navigation
Verify that all burger menu options (All Items, About, Logout, Reset App State) function correctly.

## TS-09: Cart Functionality
Verify that selected products are displayed correctly on the Cart page with accurate details.

## TS-10: Checkout Validation
Verify that error messages are displayed when required checkout fields are left empty.

## TS-11: Successful Checkout
Verify that a user can complete the checkout process successfully and see the order confirmation message.

## TS-12: Logout Security
Verify that after logging out, the user is redirected to the Login page and cannot access protected pages using the browser back button.
